Sistema de Información NETBY :: Sistema de Información NETBY :: Usuario

Description

El usuario del sistema es un actor principal del mismo, interactúa directamente con el sistema. Dentro del sistema de información se han determinado tres(3) tipos de usuario: • Administrador : accede al módulo de administración del sistema y al modulo de Capturas. Es el encargado de validar los usuarios registrados y parametrizar el mismo. • Cliente : accede a el modulo de Capturas del sistema.
