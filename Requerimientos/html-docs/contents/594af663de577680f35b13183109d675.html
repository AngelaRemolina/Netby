Description

Diagrama de casos de usos principal del sistema XYZ. El sistema de información XYZ permite a la empresa ABC limitada gestionar la información de sus producto de manera integral. Principalmente tiene tres módulos : • Módulo de administración : permite administrar los parámetros del sistema, así como los usuarios y permisos sobre el mismo. • Modulo Integrado de gestión : permite gestionar la lógica del negocio del sistema y están inmersas todas las funcionalidades relacionada con Inventarios. • Modulo de reportes : Gestiona los reportes que evidencian los resultados de cada uno de los procesos .
