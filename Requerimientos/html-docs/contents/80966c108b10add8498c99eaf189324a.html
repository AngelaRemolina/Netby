Mostrar capturas de red

UMLUseCase «RF-09»
Sistema de Información NETBY :: Módulo de Capturas :: «RF-09»Mostrar capturas de red

Description

Cada vez que un usuario ingrese al panel de control de capturas, la base de datos debe leer todos los registros de capturas generados por este cliente y esos se mostrarán en una tabla. Los usuarios de tipo Cliente podrán listar, ver el detalle de sus capturas y de las que otros clientes le compartan. (parte del CRUD de capturas en el panel de control)
